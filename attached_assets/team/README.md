# Folder Foto Tim

Silakan upload foto anggota tim di folder ini dengan format:

- `irbabul.jpg` - Foto Muhammad Irbabul Salas
- `furqon.jpg` - Foto Furqon Hanif
- `rafinda.jpg` - Foto Rafinda Mutiara Nurfitriati
- `kasnanda.jpg` - Foto Kasnanda Holymapah Siagian

**Format yang direkomendasikan:**
- Format: JPG atau PNG
- Ukuran: 300x300px (square/persegi) atau portrait
- File size: Max 500KB

Setelah upload foto, aplikasi akan otomatis menggunakan foto asli menggantikan avatar placeholder.
